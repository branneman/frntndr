PNG generation from SVG

The server.js can automatically generate fallback PNG files from your SVG images. This is done on-the-fly, and the dimensions are specified in the URL.

Say you have this file in your codebase: /static/img/arrow-ltr-black.svg

By requesting this URL, a PNG is generated from that file, with the specified dimensions:
/static/img/arrow-ltr-black.svg.10x15.png

The file will be generated and saved on disk so it will only generate when necessary.
